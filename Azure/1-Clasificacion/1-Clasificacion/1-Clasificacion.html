Clasificación¶

Las tcnicas de aprendizaje automtico supervisado implican entrenar un modelo para operar en un conjunto de caractersticas y predecir una etiqueta utilizando un conjunto de datos que incluye algunos valores de etiqueta ya conocidos. Puede pensar en esta funcin de esta manera, en la que y representa la etiqueta que queremos predecir y X representa el vector de caractersticas que usa el modelo para predecirlo.

$$y = f([x_1, x_2, x_3, ...])$$

Clasificacin es una forma de aprendizaje automtico supervisado en el que entrena un modelo para usar las caractersticas (los valores x en nuestra funcin) para predecir una etiqueta (y) que calcula el probabilidad de que el caso observado pertenezca a cada una de varias clases posibles, y prediga una etiqueta apropiada. La forma ms simple de clasificacin es la clasificacin binaria, en la que la etiqueta es 0 o 1, que representa una de dos clases; por ejemplo, "Verdadero" o "Falso"; "Interno o externo"; "rentable" o "no rentable"; y as.

Clasificación binaria¶

En este cuaderno, nos centraremos en un ejemplo de clasificacin binaria, donde el modelo debe predecir una etiqueta que pertenece a una de dos clases. En este ejercicio, entrenaremos un clasificador binario para predecir si un paciente debe o no someterse a una prueba de diabetes en funcin de algunos datos mdicos.

Las mtricas de precisin y recuperacin se derivan de cuatro posibles resultados de prediccin:

  • Verdaderos positivos: la etiqueta pronosticada y la etiqueta real son ambas 1.
  • Falsos positivos: la etiqueta predicha es 1, pero la etiqueta real es 0.
  • Falsos negativos: la etiqueta predicha es 0, pero la etiqueta real es 1.
  • Negativos verdaderos: la etiqueta pronosticada y la etiqueta real son ambas 0.

Hasta ahora, hemos considerado las predicciones del modelo como etiquetas de clase 1 o 0. En realidad, las cosas son un poco ms complejas que eso. Los algoritmos de aprendizaje automtico estadstico, como la regresin logstica, se basan en la probabilidad; entonces, lo que realmente predice un clasificador binario es la probabilidad de que la etiqueta sea verdadera (P(y)) y la probabilidad de que la etiqueta sea falsa (1 - P(y)). Se utiliza un valor de umbral de 0,5 para decidir si la etiqueta predicha es un 1 (P(y) > 0,5) o un 0 (P(y) <= 0,5). Realizar preprocesamiento en una canalización¶

En este caso, la curva ROC y su AUC indican que el modelo funciona mejor que una suposicin aleatoria, lo cual no est mal considerando que realizamos muy poco preprocesamiento de los datos.

En la prctica, es comn realizar algn preprocesamiento de los datos para que sea ms fcil para el algoritmo ajustar un modelo a ellos. Hay una gran variedad de transformaciones de preprocesamiento que puede realizar para preparar sus datos para el modelado, pero nos limitaremos a algunas tcnicas comunes:

  • Escalar caractersticas numricas para que estn en la misma escala. Esto evita que las caractersticas con valores grandes produzcan coeficientes que afecten de manera desproporcionada a las predicciones.
  • Codificacin de variables categricas. Por ejemplo, mediante el uso de una tcnica de una codificacin en caliente puede crear caractersticas binarias individuales (verdadero/falso) para cada valor de categora posible.

Para aplicar estas transformaciones de preprocesamiento, utilizaremos una caracterstica de Scikit-Learn denominada tuberas. Estos nos permiten definir un conjunto de pasos de preprocesamiento que terminan con un algoritmo. A continuacin, puede ajustar toda la canalizacin a los datos, de modo que el modelo encapsule todos los pasos de preprocesamiento, as como el algoritmo de regresin. Esto es til, porque cuando queremos usar el modelo para predecir valores a partir de datos nuevos, necesitamos aplicar las mismas transformaciones (basadas en las mismas distribuciones estadsticas y codificaciones de categoras que se usan con los datos de entrenamiento).

Nota: El trmino canalizacin se usa ampliamente en el aprendizaje automtico, a menudo con significados muy diferentes! En este contexto, lo usamos para referirnos a objetos de canalizacin en Scikit-Learn, pero es posible que vea que se usa en otros lugares para significar otra cosa.

# Train the model
from sklearn.compose import ColumnTransformer
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler, OneHotEncoder
from sklearn.linear_model import LogisticRegression
import numpy as np

# Define preprocessing for numeric columns (normalize them so they're on the same scale)
numeric_features = [0,1,2,3,4,5,6]
numeric_transformer = Pipeline(steps=[
    ('scaler', StandardScaler())])

# Define preprocessing for categorical features (encode the Age column)
categorical_features = [7]
categorical_transformer = Pipeline(steps=[
    ('onehot', OneHotEncoder(handle_unknown='ignore'))])

# Combine preprocessing steps
preprocessor = ColumnTransformer(
    transformers=[
        ('num', numeric_transformer, numeric_features),
        ('cat', categorical_transformer, categorical_features)])

# Create preprocessing and training pipeline
pipeline = Pipeline(steps=[('preprocessor', preprocessor),
                           ('logregressor', LogisticRegression(C=1/reg, solver="liblinear"))])


# fit the pipeline to train a logistic regression model on the training set
model = pipeline.fit(X_train, (y_train))
print (model)

Pruebe con un algoritmo diferente¶

Ahora probemos con un algoritmo diferente. Anteriormente usamos un algoritmo de regresin logstica, que es un algoritmo lineal. Hay muchos tipos de algoritmos de clasificacin que podramos probar, entre ellos:

  • Algoritmos de Mquina de Vectores de Soporte: Algoritmos que definen un hiperplano que separa clases.
  • Algoritmos basados en rboles: algoritmos que construyen un rbol de decisin para llegar a una prediccin
  • Algoritmos de conjunto: algoritmos que combinan los resultados de varios algoritmos bsicos para mejorar la generalizacin.

Esta vez, usaremos los mismos pasos de preprocesamiento que antes, pero entrenaremos el modelo usando un algoritmo ensamble llamado Random Forest que combina los resultados de mltiples rboles de decisin aleatorios (para obtener ms detalles, consulte Scikit -Documentacin de aprendizaje).
